# SimPoll
A simple poll app.
Requires iOS8 and later. Pick the question and choose your answer.

![alt tag](https://github.com/nemissm/simpoll/blob/master/simpoll_demo_gif.gif)

## Installation
Clone repo:

     $ git clone https://github.com/nemissm/simpoll.git

Select directory:

    $ cd path/to/simpoll
    
Install pods:

    $ pod install
